# Easy-to-use-Keyboard-Trainer
The Introduction and Beginner courses helps to learn the position of keys on the keyboard. After completing this course, you will know which finger to use to hit each key without looking at the keyboard. The course also covers upper-case letters, special symbols, and the numeric pad
